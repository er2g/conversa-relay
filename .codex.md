# Codex Notları (sunucu)

## rammfire.com “çer-çöp / indirilebilir” klasörü

- `rammfire.com` (nginx) kök path `/` → `127.0.0.1:8080` (FileBrowser).
- FileBrowser’ın servis loglarına göre root dizini: `/srv/files`.
- Yani sunucuda `/srv/files` altına koyduğum her şeyi sen `https://rammfire.com/` üzerinden indirebilirsin.

**Pratik kullanım**
- Dropzone: `/srv/files`
- Dosya bırakma örneği: `cp çıktı.zip /srv/files/çıktı.zip && chmod 644 /srv/files/çıktı.zip`
